# AI Resume Job Match Scorer

An AI-powered Resume Analyzer built using Streamlit and Groq.

## Features

* ATS Match Score
* Skill Gap Analysis
* Resume Improvement Suggestions
* Interview Preparation Tips
* Resume Rewriter
* PDF Resume Upload

## Tech Stack

* Python
* Streamlit
* Groq API
* PDFPlumber

## How It Works

1. Upload a resume PDF
2. Paste a job description
3. Get ATS-style analysis and recommendations
4. Rewrite the resume to better match the job description
